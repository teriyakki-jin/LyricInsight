## Running without Gradle Wrapper
This template does not include the Gradle wrapper JAR.  
If `./gradlew` fails, use one of the options below:

### Option A) Use IntelliJ IDEA
1. Open the `backend` folder as a project
2. Wait for Gradle import
3. Run `LyricInsightApplication`

### Option B) Install Gradle
Install Gradle 8+ and run:
```bash
gradle bootRun
```
